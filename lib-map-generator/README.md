Lib-Map-Generator
=================

A library for easy creating and managing own maps in a [JavaFX] &amp; [Maven] application.

Last release `version` is `0.1.0-SNAPSHOT` (12.2014).

Installation<a name="Installation" />
------------

TODO Update section
* If not installed download the [JRE 8] or the [JDK 8].
  * Optional: To work better with [FXML] files in a [JavaFX] application download the [JavaFX Scene Builder] under 'Additional Resources'.
* Choose your preferred IDE (e.g. [NetBeans], [Eclipse] or [IntelliJ IDEA]) for development.
* Download or clone [Lib-Map-Generator].
* Download or clone [Lib-Action].
* Download or clone [Lib-Database-ObjectDB].
* Download or clone [Lib-Logger].
* Open the project in your IDE and run it.
